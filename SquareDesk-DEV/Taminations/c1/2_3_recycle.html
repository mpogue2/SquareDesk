Recycle

At C-1, the version of this call that starts from a wave is defined to have three parts: First, the centers Fold and all adjust to make a box-circulate formation. Then all Box Counter Rotate 1/4. Then all Roll. The result is the same as the Mainstream definition.

2/3 Recycle

From a wave only: Do the first two parts of Recycle, i.e., Centers Fold (and all adjust to a box circulate formation), then Box Counter Rotate 1/4.
