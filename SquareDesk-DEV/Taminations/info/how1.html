How Taminations Works

  • Mathematics
    Each Tamination dancer is a geometric object. The dancer's movements are defined by mathematical equations.
  • Drawing
    From the equations that give a dancer's movement and position, the computer code needs to draw the circles, squares, and the rest of the graphics on the screen.
  • Writing
    There are thousands of animations collected in Taminations. Instead of writing mathematical equations for each movement, a program code makes is used to make writing dancer movements much simpler.
